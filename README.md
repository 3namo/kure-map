# 🗺️ くれウェルビーイングマップ

**呉市の住民と観光客のウェルビーイング向上を目指す総合地図アプリケーション**

呉市のオープンデータ「データプラットフォームくれ」を活用し、Wi-Fiスポット、観光地、公共施設、防災情報、イベントを統合的に可視化します。

![バージョン](https://img.shields.io/badge/version-2.0.0-blue)
![ライセンス](https://img.shields.io/badge/license-MIT-green)

---

## 🎯 コンセプト

このアプリは、呉市の「住みやすさ」と「観光体験の質」を向上させることを目的としています。

### 主なターゲット
- 👨‍👩‍👧‍👦 **呉市民**: 日常生活をより便利に、健康的に
- 🧳 **観光客**: 充実した観光体験を効率的に
- 👴 **高齢者**: バリアフリー情報で安心の外出
- 🚨 **防災意識**: いざという時の避難所・AED情報

---

## ✨ 主な機能

### 📊 1. 統合ダッシュボード
- 総スポット数、Wi-Fi利用者数、イベント情報を一目で確認
- カテゴリ別のフィルタリング機能

### 🏯 2. 5つのカテゴリ表示
1. **📶 Wi-Fiスポット** (23地点)
   - 利用者数に応じた視覚化
   - 混雑状況の把握

2. **🏯 観光スポット** (8地点)
   - 大和ミュージアム、てつのくじら館など
   - 営業時間、料金、所要時間、バリアフリー情報

3. **🏢 公共施設** (6施設)
   - 図書館、体育館、病院、子育て支援センター
   - 開館時間、混雑度、Wi-Fi有無

4. **🎉 イベント** (5件)
   - 呉みなと祭、海自カレーフェスタなど
   - 開催日程、場所、料金情報

5. **🚨 防災・安全** (9施設)
   - 避難所、AED、警察署、消防署
   - 緊急時の迅速な情報アクセス

### 🔍 3. 検索機能
- 施設名や住所での検索
- リアルタイムフィルタリング

### 🎨 4. 直感的なUI/UX
- カテゴリ別の色分け表示
- クイックフィルターボタン
- レスポンシブデザイン（PC/タブレット/スマホ対応）
- サイドバー折りたたみ機能

### ♿ 5. アクセシビリティ
- バリアフリー対応施設の表示
- 見やすい凡例とアイコン
- キーボード操作対応

## 🚀 使用方法

### ローカルで実行

1. **リポジトリをクローン**
```bash
git clone https://github.com/3namo/kure-map.git
cd kure-map
```

2. **ローカルサーバーを起動**

**注意**: `file://` プロトコルでは動作しません。JSONファイルの読み込みにHTTPサーバーが必要です。

#### Python 3の場合:
```bash
python -m http.server 8000
```

#### Python 2の場合:
```bash
python -m SimpleHTTPServer 8000
```

#### Node.jsの場合:
```bash
npx http-server -p 8000
```

3. **ブラウザで開く**
```
http://localhost:8000
```

## 📱 使い方

### 基本操作
1. **サイドバー**: 統計情報、検索、カテゴリフィルター
2. **地図**: マーカーをクリックで詳細情報を表示
3. **クイックフィルター**: 上部のボタンで素早くカテゴリ切り替え
4. **凡例**: 右下でマーカーの意味を確認

### フィルタリング
- ✅ チェックボックスまたはクイックボタンでカテゴリ表示/非表示
- 🔍 検索バーで施設名・住所検索

### レスポンシブ対応
- 📱 スマホ: サイドバーは自動折りたたみ、ハンバーガーメニューで開閉
- 💻 PC: サイドバー常時表示、広い地図表示エリア

## 📊 データ形式

### Wi-Fiスポット (wifi-data.json)
```json
{
  "2025/10/01": [
    {
      "Wifi名": "施設名",
      "設置場所住所": "住所",
      "緯度": "34.xxxxx",
      "経度": "132.xxxxx",
      "利用者数": "100"
    }
  ]
}
```

### 観光スポット (tourism-spots.json)
```json
{
  "spots": [
    {
      "id": "tour_001",
      "name": "施設名",
      "category": "観光スポット",
      "type": "museum",
      "description": "説明文",
      "address": "住所",
      "latitude": "34.xxxxx",
      "longitude": "132.xxxxx",
      "openingHours": "9:00-18:00",
      "admission": "500円",
      "tags": ["人気", "ファミリー"],
      "barrierFree": true
    }
  ]
}
```
